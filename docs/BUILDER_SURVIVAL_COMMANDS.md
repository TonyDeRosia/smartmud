# Builder Survival Commands

Phase 11D2 survival Builder/Admin content includes needs, consumables, rest quality, rest locations, sleep profiles, campfire profiles, fuel profiles, and campsite profiles. Validation checks bounded need values, valid references, positive rest/camp durations, valid quality bounds, and conservative warning conditions.

Runtime diagnostics include `needstrace`, `consumptiontrace`, `resttrace`, `campfiretrace`, `campsitetrace`, and `survivalaudit`. Runtime mutations remain Admin/Owner responsibilities.
