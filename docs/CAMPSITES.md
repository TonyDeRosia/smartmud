# Campsites

Campsites are temporary room-bound runtime objects for Phase 11D2. Profiles live in `campsite_profiles`; runtime state lives in `campsite_instances`. Campsites can be created, occupied, expired, dismantled, or archived without creating permanent structures or terrain ownership.

Commands include `camp`, `camp here`, `make camp`, `camp status`, `break camp`, and `inspect campsite`. Starter content provides `basic_campsite`, which uses `bedroll_rest` for rest location behavior.
